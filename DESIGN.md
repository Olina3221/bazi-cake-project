# Laiten 萊點 ── 品牌設計系統

## 品牌概述
- **品牌名稱**：Laiten / 萊點
- **品牌口號**：山交手作，溫度共享
- **產業**：台灣手作甜點外燴、八字命格客製蛋糕、企業下午茶
- **風格關鍵字**：極簡、溫暖、手感、留白、精緻

---

## 色彩系統

| 名稱 | 色碼 | 用途 |
|:--|:--|:--|
| 米白（主背景） | `#FAF8F3` | 頁面底色 |
| 暖米（次要背景） | `#F0E8D8` | 卡片、tag、hover 底色 |
| 暖金棕（強調色） | `#C4975A` | 按鈕 hover、價格、重點標示 |
| 深棕黑（主文字） | `#2C1A0E` | 標題、主要文字、深色按鈕 |
| 中棕（次要文字） | `#7A5C40` | 說明文字、副標題 |
| 邊框線 | `#E5D9C8` | 分隔線、卡片邊框 |

**配色原則**：
- 背景以米白和留白為主，不用純白 `#FFFFFF`
- 強調色（暖金棕）只用在關鍵互動元素，不要大面積使用
- 避免高彩度顏色（鮮紅、電藍、螢光）
- 深色 CTA 區塊用 `#2C1A0E`（深棕黑）而非黑色 `#000000`

---

## 字體

| 用途 | 字體 | 備註 |
|:--|:--|:--|
| 標題、品牌名 | Noto Serif TC | Google Fonts，載入 weight 300/400/500 |
| 內文、按鈕 | Noto Serif TC | 同字體族，保持一致性 |
| 英文 | Noto Serif TC | 中英混排皆用此字體 |

**字體原則**：
- 標題用 `font-weight: 300`（細體），有精緻感
- 字間距（letter-spacing）偏大，`0.05em ~ 0.15em`
- 不用粗體標題（除非強調）

## 元件樣式

### 按鈕
```css
/* 主要 CTA（淺底深字） */
border: 1px solid #2C1A0E;
color: #2C1A0E;
background: transparent;
padding: 10px 24px;
/* hover: 反色 */

/* 深色 CTA 區塊（深底淺字） */
border: 1px solid #FAF8F3;
color: #FAF8F3;
background: transparent;
/* hover: 反色 */
```
- 按鈕用「框線按鈕」，不用實心填色按鈕
- 圓角極小或不圓角（0px ~ 2px）

### 卡片
```css
background: #FFFFFF;
border: 1px solid #E5D9C8;
/* 無陰影或極淡陰影 */
```

### 標籤 / Badge
```css
background: #F0E8D8;
color: #7A5C40;
font-size: 11px;
letter-spacing: 0.05em;
padding: 3px 8px;
```

## 排版原則

- **留白**：大量留白，section 間距 `80px ~ 120px`
- **最大寬度**：`max-width: 1024px`，內容不要鋪滿全寬
- **文字對齊**：Hero 文字置中，其他區塊靠左
- **行距**：內文 `line-height: 1.8`，標題 `line-height: 1.4`

---

## 風格守則

**Do（該做的）**：
- 大量留白，讓每個元素都有呼吸空間
- 用細字體傳遞精緻感
- 圖片比例統一（`4:3` 產品圖）
- 文案語氣溫柔、有詩意，避免促銷感

**Don't（不該做的）**：
- 不用漸層背景
- 不用投影陰影（box-shadow）
- 不用多種顏色混搭
- 不用圓角過大的按鈕（像 2015 年的網站）
- 不用全大寫英文標題（違反手感質感）
