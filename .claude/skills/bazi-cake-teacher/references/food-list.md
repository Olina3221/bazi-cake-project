## 底色規則

| 缺五行 | 底色選項 |
|--------|----------|
| 缺木 | 米白、奶油白 |
| 缺火 | 暖白、珊瑚粉、蜜桃橘 |
| 缺土 | 奶茶色、焦糖米 |
| 缺金 | 純白、珍珠白、奶白 |
| 缺水 | 淡霧藍、淡紫灰、珍珠灰白 |

## 夾層食材清單

夾層共兩層，對應地支藏干裡需要補的五行。
**夾層食材只用在夾層，不用在秀面。**

### 木系夾層 🌿
- 檸檬凝乳

### 火系夾層 🔥
- 草莓果醬
- 覆盆子庫利凍
- 草莓庫利凍
- 火龍果果凍
- 紅心火龍果
- 玫瑰鮮奶油
- 櫻桃果醬

### 土系夾層 🏔️
- 芋頭泥
- 焦糖奶油
- 布丁
- 南瓜泥（秋冬）
- 栗子泥（秋冬）

### 金系夾層 ✨
- 香草籽鮮奶油
- 白桃果醬
- 奶酪

### 水系夾層 💧
- 藍莓庫利凍
- 藍莓果醬
- 小藍莓罐頭（瀝水）
- 黑芝麻鮮奶油（黑芝麻粉製作，只用夾層不用秀面）

## 秀面食材清單

**重要：輸出前依當季月份過濾，只列可用選項。**

### 木系秀面 🌿（生長、新生、突破）

| 季節 | 食材 |
|------|------|
| 常備 | 抹茶碎、檸檬皮絲、薄荷葉 |
| 春（2-4月） | 桑椹（四月限定）、青葡萄 |

| 冬（12-1月） | 檸檬、萊姆、橘子、柳橙 |

| 冬（12-1月） | 草莓、紅棗（備用） |
